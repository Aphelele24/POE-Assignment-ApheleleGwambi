# ApheLux Boutique Website

## 📌 Project Overview

ApheLux is an online boutique website designed to showcase stylish and affordable fashion for young women. The website aims to provide a clean, modern, and user-friendly shopping experience.

---

## 🎯 Purpose

The purpose of this project is to:

* Promote a boutique brand
* Display fashion products attractively
* Practice front-end web development skills using HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (External stylesheet)
* JavaScript (External file)
* Google Fonts (Playfair Display)

---

## ✨ Features

* Navigation bar with multiple pages
* Promotional sale banner
* Product image gallery
* “Shop Now” button linking to services
* Embedded video section (click-to-play, muted)
* Custom play button overlay
* Responsive layout improvements

---

## 🎥 Video Feature

The homepage includes a “See It In Action” section:

* Video is embedded using the `<video>` tag
* Video is muted for better user experience
* Custom play button activates video using JavaScript
* Video styled to be short and wide using CSS
